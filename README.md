# Daily Shift Instruction Login Page

## Project Description

Daily Shift Instruction Login Page is a secure and responsive authentication interface designed for a shift management and instruction system.
The system allows employees, operators, or administrators to log in and view daily shift instructions, operational updates, and system information.

This project provides a modern login UI that can be integrated with a backend system for authentication and shift data management.

---

## Technologies Used

* React JS
* Node.js
* Express.js
* MongoDB
* CSS3
* REST API

---

* Secure Login Interface

* Shift Management System Access

## Installation

Clone the repository

```
git clone https://github.com/yourusername/daily-shift-instruction-login.git
```

---

### Install Frontend

```
cd frontend
npm install
npm start
```

---

### Install Backend

```
cd backend
npm install
node server.js
```
